## 核心逻辑（历史版本）

通过环境温度影响函数对温差进行空间投影，结合电流权重计算加权温差，与同类电池箱线图上边缘比较。

- 温度影响函数：f(T) = (0.5 / (e^(-A×T1) + 0.5)) × T2，A = ap11_temp_factor
- 电流影响函数：f(I) = e^(-B×|I|)，B = ap11_current_factor
- 判定：单电池温差加权Q25 > 同类电池上边缘
